# Preset — Minimalist / editorial (Linear, Notion, Vercel)

Load this preset only if the brief implies restrained, text-forward, editorial. It overrides
the dials toward `VARIANCE 3 / MOTION 3 / DENSITY 5`. Foundations rules (`10`–`14`, a11y `20`)
still apply — this is a direction, not an exemption.

## The look

Restraint as the statement. Monochromatic or near-mono palette, one accent used sparingly.
Type does the work: strong hierarchy from weight and size, generous but purposeful space, hair-
line borders over shadows. Nothing decorative earns its place unless it clarifies.

## Rules

1. **Near-monochrome.** Neutral scale (tinted, not dead gray, `11`) + exactly one accent, used
   for the primary action and little else. No gradients.
2. **Type carries hierarchy.** Weight + size + color, not boxes and color blocks. One strong
   text family; a display cut allowed if genuinely editorial.
3. **Space is structural.** Consistent, generous rhythm (`13`); alignment is strict; the grid is
   visible in the calm.
4. **Hairline borders, minimal shadow** (`14`). Depth from layering and space, not elevation.
5. **Motion is subtle and functional** (`40`): quiet transitions, no reveals-for-drama.
6. **Content-first.** Remove chrome; let the content be the interface. High signal-to-noise.

## Avoid

Gradients, glassmorphism, decorative illustration, multiple accents, heavy shadows, big motion.
Minimalism fails when it becomes *empty* — restraint still needs clear hierarchy and obvious
primary actions. Sparse ≠ unclear.
